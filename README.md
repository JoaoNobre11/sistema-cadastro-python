# Sistema de Cadastro de Usuários

## 📌 Descrição
Projeto simples em Python para cadastrar e listar usuários via terminal.

## ⚙️ Tecnologias
- Python

## 🚀 Funcionalidades
- Cadastrar usuário
- Listar usuários
- Armazenamento em arquivo (.txt)

## ▶️ Como executar
1. Baixe o projeto
2. Execute o comando:

python main.py

## 📈 Objetivo
Projeto criado para praticar lógica de programação e manipulação de arquivos.
